dataset.csv is where you say which text file is connected to which bitmap file.
The file paths you use will be relative to bin/input for the text, and bin/output for the bitmap.
If you want to use asian characters, be sure to encode your file as UTF-8. ("Encoding->Encode in UTF-8" in notepad++)
